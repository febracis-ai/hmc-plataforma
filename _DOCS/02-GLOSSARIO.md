## Marca e identidade

| Termo | Definição |
|---|---|
| **HMC — High Mentor Club** | Nome oficial. **Singular "Mentor"** deliberado (padrão Febracis CIS/FCIS/IEX, 3 letras). Sigla + lockup sempre juntos em peças oficiais. |
| **High Mentor** (identidade) | Como cada membro se autodenomina. Cada membro **é UM** High Mentor — não "um dos High Mentors". Análogo a "Apple Genius". |
| **The Club / Os High Mentors** | Forma coletiva. Usar "The Club" em registro institucional; "Os High Mentors" quando o coletivo for sujeito da frase. |
| **The Curator / O Curador** | Paulo Vieira no papel institucional. **Não sócio**, não mentor individual, não vendedor. Escolhe, estabelece padrão, mantém coesão, convoca, conecta. |
| **MELID / Mentor de Líderes** | Programa-mãe de 12 meses, R$ 75k renovação anual. **HMC é continuidade institucional do MELID.** |
| **Federação Febracis** | Ecossistema-mãe (MELID, HMC, Portal Franqueado, DRE). |

| **II** | **High Mentor** | R$ 56.000 a R$ 85.000 | Prata fosca `#B8B5B0` | Identidade central do clube; voz audível no Council; posicionamento de destaque em eventos; prioridade no Cross-Floor |

## Estrutura ritual do clube

| Termo | Definição |
|---|---|
| **HMC Council** | Sessão mensal do clube. Online padrão, presencial em alguns meses. **Cadência: mensal.** Hotseats + curadoria do Curador na segunda metade. |
| **The Curator's Address** | PGR mensal aberto pelo Curador, online ao vivo, ~2h, disponível para **todos os mentorados de todos os membros**. Multiplica marca do membro. |
| **The Antechamber** | Grupo WhatsApp direto com o Curador. Acesso a membros ativos. **Regras tácitas:** sem bom-dia coletivo, sem aniversariante, sem pergunta de aluno. |
| **Cross-Floor** | Marketplace interno de vagas cruzadas. **20% das vagas** de cada membro circulam para mentorados de outros membros. Metáfora "crossing the floor" parlamentar britânico. |
| **HMC Success Fee** | Comissão de **16%** sobre vendas do Método CIS referidas pelo membro. Link único por High Mentor + dashboard + pagamento automatizado Febracis. |
| **The Bridge** | Pipeline curado de migração de mentorados às **franquias do ecossistema PV**. **Exclusivo Tier III.** Não é venda — é "travessia institucional" mediada pelo Curador. |
| **The Pilgrimages** | Os 3 eventos premium anuais. "Peregrinação" — exige deslocamento, preparação, intenção. |
| **The House** | Peregrinação americana, julho. Acesso restrito a quem fez o **Revolution**. Hospitalidade direta do Curador. Propósito: aproximar o membro do Curador como pessoa — não como autoridade. |
| **Inter Business** | Peregrinação corporativa, final do ano. Formato e local anunciados anualmente. Propósito: encontro institucional do clube onde decisões coletivas do ano são fechadas e o seguinte é desenhado. |
| **Revolution** | Experiência imersiva exclusiva de transformação comportamental e alta performance ministrada por Paulo Vieira. **Externo ao HMC.** Aparece como pré-requisito de admissão a The House. |
| **HMC Summit** | Encontro anual internacional do clube. Local definido anualmente pela curadoria. Ápice do calendário ritual. |
| **The High Mentor Codex** | Anuário anual do clube. Vol I = **Founding Edition** (registra a casa nascendo). Edições futuras catalogam membros por cadeira. Composto em Cormorant + Agharti + Poppins. Dourado fosco sobre Noite Profunda. |
| **HMC Seal / High Mentor Mark** | **Chancela circular** emitida ao membro para uso em seu produto. **O ativo mais importante da identidade.** 3 variações por tier (bronze/prata/dourado). |
| **Cadeira (Seat)** | Posição numerada do membro (Cadeira I, XII, etc.). **Em romanos.** Aplicada no rodapé do selo. Cadeiras fundadoras (I–XX) registradas perpetuamente no Codex Vol. I. |
| **Founding Edition** | Vol. I do Codex, publicado junho 2026. Registra origem, não membros. |
| **The Founding Members** | Primeiros High Mentors admitidos no ciclo 2026. Cadeira vitalícia simbólica no Codex Vol. I. |

| **Sofia** | Segunda curadora de admissão. |
| **Mentores externos candidatos** | Operam mentoria própria; entram por convite + avaliação; não passaram pelo MELID, mas aceitam o Método CIS como língua. Porta II. |
| **Alunos do MELID** | Não-mentores em formação. Ao concluir + renovar, podem ser convidados ao HMC. Porta I. |
| **Mentores responsáveis pelos 4 Motores** | Bruno Vaz (Funil), Júlia Vieira (PMN), responsável YouTube Poderoso (a definir), PV (Método Referido). **Mentores técnicos**, NÃO membros do clube. |
