Esta página NÃO existe em produção. Em prod, login = Supabase Auth com 2FA obrigatório, IP allowlist no firewall (Cloudflare Worker `apps/edge`), JWT validado pelo middleware NestJS e RLS no Postgres aplicando role tier-based. Multi-persona como Lucas (Mentor + Member + Mentorado simultaneamente) é nativo do schema — uma única conta, múltiplos roles. Mode-switch é renderizado server-side com base em user.capabilities · membros sem permissão veem 403 ao tentar acessar rota inválida.
